## 电商平台实时监控系统

## 一、技术栈

该项目主要使用 `Vue`搭配`Echarts`采用组件化的方式开发完成,此外项目中陆续使用有
- VueRouter
- VueX
- WebSocket
- axios/fetch
- vue-cli
- ES6
- less
